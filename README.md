## **The Vision** 
The main purpose of the website is to function as an informative and interactive platform dedicated to fragrances, designed to guide users through the process of understanding and exploring perfumes in a way that feels approachable rather than overwhelming. Instead of functioning as a simple product display, the site is intended to act as a learning space where users can familiarize themselves with scent families, fragrance notes, and essential terminology before making choices. The intention is to close the gap between curiosity and informed decision-making by presenting fragrance knowledge in a way that is both accessible to beginners and still engaging for more experienced enthusiasts.  In essence, the website aims to make fragrance discovery less intimidating and more intentional, so users can choose scents that genuinely align with their preferences, identity, and lifestyle.
